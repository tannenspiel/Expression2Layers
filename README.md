# Property Expression Tool for After Effects

**Property Expression Tool** — это утилита для Adobe After Effects, позволяющая быстро копировать и массово применять выражения (expressions) к аналогичным свойствам других слоёв. Ускоряет пакетную работу со свойствами трансформации и пользовательскими параметрами.

## Возможности

- Получить выражение из выбранного свойства слоя.  
- Вставить и применить это выражение ко множеству выделенных слоёв.  
- Поддержка как стандартных трансформационных параметров (`Position`, `Scale`, `Rotation`, `Opacity`, `Anchor Point`), так и пользовательских свойств эффектов.  
- Встроенный интерфейс для просмотра и редактирования текста выражения.  

## Как использовать

1. Откройте композицию и выберите слой, у которого есть нужное свойство.  
2. Выделите свойство в панели **Timeline** (например, `Transform > Opacity`) и нажмите кнопку **Get Custom Property**.  
   Скрипт определит имя свойства и добавит его в список.  
3. В поле ввода **Expression Text** появится область для вставки или редактирования выражения.  
4. Выберите одно или несколько слоёв, на которые нужно применить выражение.  
5. В списке свойств выберите нужное (например, `Position` или `Custom: My Slider`).  
6. Нажмите кнопку **WRITE EXPRESSION to Selected Layers**.  
   Скрипт применит указанное выражение ко всем выделенным слоям, где найдено аналогичное свойство.  

## Интерфейс

- Поле `Expression Text` — ввод и редактирование выражения.  
- Кнопка **Get Custom Property** — запоминает выделенное свойство активного слоя.  
- Список свойств — содержит стандартные трансформации и последнее выбранное свойство.  
- Кнопка **WRITE EXPRESSION to Selected Layers** — применяет текущее выражение к выбранным слоям.  

## Установка

1. Сохраните файл как `PropertyExpressionTool.jsx`.  
2. Поместите его в папку:
%USERPROFILE%\Documents\Adobe\After Effects <версия>\Scripts\ScriptUI Panels\
3. Перезапустите After Effects.  
4. Откройте через меню: **Window → Property Expression Tool**.  

## Требования

- Adobe After Effects CC 2020 или новее.  
- Режим **Allow Scripts to Write Files and Access Network** должен быть включён в настройках.
